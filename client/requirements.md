## Packages
recharts | Data visualization charts (Bar, Pie, Line)
@tanstack/react-table | Headless UI for building complex tables
framer-motion | Animation library for smooth transitions
date-fns | Date formatting utilities
clsx | Class name utility
tailwind-merge | Class name merging utility

## Notes
- The app uses a dashboard layout with a sidebar.
- File uploads are handled via FormData to /api/upload.
- Analytics data is fetched via GET requests.
- Visual style is "Government Professional" - clean, high contrast, blue/gray palette.
